使用说明

服务器API地址

前缀： http://xxx

完整的API地址为：前缀+具体接口路径

比如，获取账户信息接口为： http://xxx/ + /api/v1/getAccount?name=prefsabi@homes

->


## 调用接口说明
- 如果参数格式是JSON的话：提交request请求时必须添加header头：Content-Type:application/json


+ 所有的接口的返回形式都是统一为：

 正常返回:

{ "errorCode": 0, "errorMsg": "", "data": 某种类型的数据，比如字符串，数字，字典等等 } ``` 异常返回: 

{
  "errorCode": 具体的错误码,
  "errorMsg": "具体的错误信息字符串"
  "data": null
}

  • URL: /api/v1/getAccount?name={name}
